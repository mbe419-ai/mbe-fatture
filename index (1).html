⚙️ Configurazione
Ottenuto da Google Apps Script → Distribuisci → Gestisci distribuzioni
Identifica questa postazione nel log delle modifiche
L'app controlla automaticamente se altre postazioni hanno inviato dati
📋 Istruzioni rapide
1. Apri Google Sheets → crea un foglio qualsiasi → Estensioni → Apps Script
2. Elimina tutto il codice presente e incolla il file MBE_AppsScript.gs
3. Clicca 💾 Salva → ▶ Esegui → seleziona inizializza → autorizza → esegui
Questo crea il file MBE_FatturePassive_DB.json sul tuo Google Drive
4. Distribuisci → Nuova distribuzione
Tipo: App Web · Esegui come: Me · Chi ha accesso: Chiunque
5. Copia l'URL e incollalo nel campo qui sopra
6. Inserisci nome postazione → 💾 Salva → 🔌 Testa connessione
7. Ripeti la configurazione (stesso URL) su ogni altra postazione
💡 I dati vengono salvati nel file MBE_FatturePassive_DB.json nel tuo Google Drive — visibile e scaricabile da Drive in qualsiasi momento.
